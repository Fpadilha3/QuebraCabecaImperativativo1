Frontend Inteligente com JavaScript

🎯 Objetivo

Nesta aula, você vai evoluir o projeto aprendendo a trabalhar com múltiplos elementos e estados dinâmicos.

Também vai entender como o JavaScript pode controlar o comportamento visual da interface.

🧠 Revisão

O DOM representa a estrutura da página no navegador.

Com JavaScript, conseguimos acessar e modificar elementos.

Eventos permitem reagir às ações do usuário.

🚀 Manipulação de Classes


elemento.classList.add("ativa");
elemento.classList.remove("ativa");
elemento.classList.toggle("ativa");

O toggle alterna automaticamente o estado.

🧠 Estado Dinâmico


if (elemento.classList.contains("ativa")) {
  // está ativo
}

📦 Trabalhando com Vários Elementos


const pecas = document.querySelectorAll(".peca");

🖱️ Eventos em Múltiplos Elementos


pecas.forEach(function(peca) {

  peca.addEventListener("click", function() {
    // ação ao clicar
  });

});

🧩 Evolução do Projeto

• várias peças funcionando ao mesmo tempo

• interação direta

• mudança visual dinâmica
